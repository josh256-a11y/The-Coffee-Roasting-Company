# Coffee Roasting Company Website

This repository contains the full website for *The Coffee Roasting Company*,
including responsive layout, gallery, form validation, and a custom lightbox plugin.

## Features
- Coffee-themed colour palette (coffee brown, gold, green)
- Fully responsive layout (desktop, tablet, mobile)
- Google Maps location (1 Gold Street, Martindale, Johannesburg)
- Modern image gallery with professional-style lightbox
- Form validation for all contact/enquiry forms
- SEO-friendly meta tags
- Organized folder structure (CSS, Javascript, Images, Pages)

## Lightbox Plugin
A custom-built professional lightbox plugin is included:
- Click any image to enlarge
- Click outside or close button to dismiss
- Smooth shadows, rounded corners

## Folder Structure
```

Javascript/lightbox.js

```

## GitHub Instructions
To upload this project to GitHub:

### 1. Create a new repository
- Go to https://github.com
- Click **New Repository**
- Name it: `coffee-roasting-company` (or your choice)

### 2. Upload Files
Option A: Drag and Drop  
Drag the entire folder contents into GitHub's web uploader.

Option B: Command Line
```
git init
git add .
git commit -m "Initial upload"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
git push -u origin main
```

### 3. How to Check Your GitHub Repo Link
- Go to your repository page
- Click the green **Code** button
- Copy the HTTPS URL (Example):
```
https://github.com/username/coffee-roasting-company.git
```
